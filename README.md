# Registration Form

A simple and responsive **Registration Form** built using **HTML, CSS, and JavaScript**. This project demonstrates client-side form validation with real-time feedback to improve the user experience.

## 🚀 Features

* User-friendly registration form
* Real-time input validation
* Name field validation (cannot be empty)
* Email format validation
* Password validation (minimum 6 characters)
* Error messages displayed instantly
* Submit button remains disabled until all fields are valid
* Responsive and clean user interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

## 📋 Validation Rules

* **Name:** Required and cannot be empty.
* **Email:** Must be in a valid email format (e.g., [example@email.com](mailto:example@email.com)).
* **Password:** Must contain at least **6 characters**.
* **Submit Button:** Enabled only when all fields pass validation.

## ▶️ How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Double-click `index.html` or open it in your preferred web browser.
4. Fill in the form and observe the real-time validation.
